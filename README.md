# Controle de Pedidos e Clientes 📱

Projeto Android desenvolvido utilizando **Kotlin** e **Jetpack Compose**.

O aplicativo tem como objetivo realizar o gerenciamento de clientes, produtos e pedidos, utilizando uma interface moderna e recursos de persistência de dados.

## Funções principais

- Login com usuário armazenado no banco local;
- Cadastro, edição e exclusão de clientes;
- Cadastro, edição e exclusão de produtos;
- Cadastro, edição e exclusão de pedidos;
- Relacionamento entre clientes, produtos e pedidos;
- Armazenamento de configurações utilizando DataStore;
- Navegação entre telas utilizando Navigation Compose;
- Arquitetura organizada utilizando ViewModel e StateFlow;
- Interface desenvolvida com Jetpack Compose e Material Design.

## Como abrir o projeto

1. Extraia o arquivo ZIP do projeto.

2. Abra a pasta do projeto no Android Studio.

3. Aguarde a sincronização do Gradle.

4. Execute o aplicativo em um emulador ou dispositivo Android.

5. Utilize um dispositivo com **Android API 26 ou superior**.

---

## Observações

O arquivo `local.properties` não está incluído no projeto, pois ele é criado automaticamente pelo Android Studio de acordo com o caminho do SDK configurado em cada computador.
